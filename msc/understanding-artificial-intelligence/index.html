Unit 7: Natural Language Processing (NLP)

Synopsis

Introduces NLP, challenges in understanding human language, and key techniques such as Word2Vec.

Main objectives

  • Understand basic NLP concepts and challenges.
  • Compare different NLP approaches and technologies.

Unit 8: Understanding Natural Language Processing (NLP)

Synopsis

Deepens NLP understanding with Word2Vec demos and creating parse trees.

Main objectives

  • Explain key elements of NLP models.
  • Engage with worked examples like Word2Vec.
  • Create constituency-based parse trees.

Unit 9: Introduction to Adaptive Algorithms

Synopsis

Introduces artificial neural networks and deep learning, focusing on their core ideas and strengths/weaknesses.

Main objectives

  • Understand basic ANN and deep learning concepts.

Unit 10: Deep Learning in Action

Synopsis

Explores real-world applications of deep learning, with focus on ethical and socio-economic implications.

Main objectives

  • Analyse real-world deep learning systems.
  • Consider ethical, social, and economic impacts.
  • Understand data’s role in these technologies.

Unit 11: Intelligent Agents in Action

Synopsis

Applies concepts to Industry 4.0 and FinTech, and focuses on delivering the individual development project.

Main objectives

  • Apply agent and deep learning concepts in specific sectors.
  • Evaluate efficiency and impact of intelligent technologies.
  • Implement and test an intelligent agent system in Python.
